# 深中飞跃手册

**深中飞跃手册** 是一个来自深圳中学往届学长学姐的 **高考录取案例与就读体验参考库**，  
面向正在进行院校选择的学弟学妹，提供 **真实、结构化、可检索** 的参考信息。

请在每次使用本网站前，查看首页中的 **「免责声明」** 。

## 这个网站包含什么？

本手册中的所有案例均来自校友投稿，内容包括：

- 高考分数与排名
- 录取院校与录取专业
- 对院校与专业的真实评价
- 给学弟学妹的个人建议

---

## 如何使用？

你可以通过以下方式浏览案例：

- **案例总览**：查看所有已收录案例  
- **按院校查看**：汇总某一院校的校友评价  
- **按专业查看**：汇总某一专业的校友评价
- **经验分享**：汇总所有案例的经验
- **学长学姐说**：查看学长学姐投稿的文章

页面右上角支持 **全文搜索**，可直接在所有案例中搜索：院校名 / 专业名 / 分数 / 昵称

---

## 如何投稿？

如果你是深中校友，欢迎你参与共建本手册。

- 投稿内容支持 **匿名**
- 所有案例在展示前会进行基本整理
- 未填写的选填项不会影响展示

投稿入口请见导航栏中的 **「我要投稿」**
